1. Hugo 和 Go 的下载与安装

hugo 是基于 go 的，因此为了使用 hugo 创建个人博客，首先需要下载go和hugo，hugo的quickinstall 里面详细介绍了关于 go 和 hugo 在不同平台的下载和安装。这里列出一些相关的连接，并不对安装进行过多的记录。

2. 开始使用 hugo

创建新网站

hugo new site mysite

选择一个主题

上述的命令即利用hugo创建了一个个人站点，并命名为 mysite，接着我们从 Hugo 的主题 中找到一个喜欢的主题，下载放进 theme/ 文件夹中。通常把 exampleSite 文件夹中的配置文件复制到 mysite 中。

这里推荐找一个文档比较齐全的主题，这里我选择了 eureka 这个主题，功能比较齐全，且文档说明比较详细。

启动hugo查看site

命令行键入：

hugo server -D

git p 可以得到 localhost:1313 的地址，从浏览器打开变可以看到自己的 site 了。


3. 在 Github 上部署 hugo

创建仓库

首先在 Github上面创建一个 repository 要和自己的 ID 一致。

注意你的仓库里不要有，也不要创建 readme.md 文件，否则无法显示网页。

建立静态页面

利用 hugo 建立静态页面

hugo --theme=${your theme} --baseUrl="https://${yourID}.github.io" -D

上传到 github

cd public
git init
git add.
git commit -m "${your commit}"
git remote add orgin ${your https of github}
git pull --rebase origin master
git push origin master
